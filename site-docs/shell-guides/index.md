# Shell Guides

This section mirrors the documentation surfaces that are available from the
interactive shells and wrapper menus.

## What is covered here

- the SCP03 `GUIDE [Topic]` content that is normally shown inside the admin shell
- the SCP03 `HELP` command reference grouped the same way as the terminal help surface
- the relationship between the in-shell guide wizard and the authored Markdown documents in the repository

## Where these come from

The main wrapper exposes these from the Guides menu, and the SCP03 admin shell
exposes them directly in-session:

- `GUIDE`
- `GUIDE GP`
- `GUIDE ETSI`
- `GUIDE GSMA`
- `GUIDE INSTALL`
- `GUIDE SECURITY`
- `GUIDE OTA`
- `GUIDE CONFIG`
- `GUIDE SAIP`
- `GUIDE SUCI`
- `GUIDE CLI`
- `HELP`
