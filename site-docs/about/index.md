# About YggdraSIM

YggdraSIM is a Python toolkit for secure-element research, eUICC analysis,
SIM and eSIM management, SCP11 relay and local flows, SCP80 OTA, HIL
bridging, and SAIP profile-package work.

## Contributing

External contributions are welcome under the project license and the
constraints described in [Contributing](contributing.md). There is no
CLA. By submitting a pull request you agree that your contribution is
licensed to the project under `GPL-3.0-or-later` and may be
redistributed by 1oT OÜ as part of YggdraSIM.

For coordinated vulnerability disclosure, follow the process in
[`.github/SECURITY.md`](https://github.com/1oT/YggdraSIM/blob/main/.github/SECURITY.md)
rather than opening a public issue.
